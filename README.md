SEMINARIO DE PYTHON - Actividad 2

Este proyecto resuelve los ejercicios de la Actividad 2 de Python.
Las funciones estan organizadas en la carpeta src y se utilizan desde un notebook de Jupyter.

- Estructura del proyecto

act2-python/
│
├── notebooks/
│   └── actividad2.ipynb
│
├── src/
│   └── funciones.py
│
├── README.md
├── .gitignore
└── requirements.txt

- Requisitos
* Python 3.12.9
* JupyterLab

- Instalación 
1. Clonar el repositorio:

git clone https://github.com/santip-04/act2-python.git

2. Entrar en la carpeta del proyecto:

cd act2-python

3. Instalar dependencias:

pip install -r requirements.txt

- Ejecución

1. Iniciar JupyterLab:

jupyter lab

2. Abrir el archivo:

notebooks/actividad2.ipynb

3. Ejecutar las celdas.
